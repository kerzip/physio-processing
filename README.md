# physio-processing

Collection of processing steps to analyse physiological data recordings.

One recording data stream includes [labstreaminglayer](https://github.com/labstreaminglayer) streams from following apps:
- [BrainVisionRDA](https://github.com/labstreaminglayer/App-BrainProducts/tree/8373006a351a1f84d82f598ce0abb723caac16d8/BrainVisionRDA)
- [gUSBamp](https://github.com/labstreaminglayer/App-g.Tec/tree/1f35f71d0a70ced7ea4025409293587cce7102cb/g.USBamp)
- [AppPupilLabs](https://github.com/labstreaminglayer/App-PupilLabs)

Processing of data is done with [NeuroKit](https://github.com/neuropsychology/NeuroKit.py), [MNE](https://mne.tools/stable/index.html) and [BioSPPy](https://github.com/PIA-Group/BioSPPy)


For more information see:
BioSPPy : Carreiras C, Alves AP, Lourenço A, Canento F, Silva H, Fred A, et al. BioSPPy - Biosignal Processing in Python, 2015

MNE: A. Gramfort, M. Luessi, E. Larson, D. Engemann, D. Strohmeier, C. Brodbeck, R. Goj, M. Jas, T. Brooks, L. Parkkonen, M. Hämäläinen, MEG and EEG data analysis with MNE-Python, Frontiers in Neuroscience, Volume 7, 2013, ISSN 1662-453X

NeuroKit: Makowski, D. (2016). NeuroKit: A Python Toolbox for Statistics and Neurophysiological Signal Processing (EEG, EDA, ECG, EMG...). Memory and Cognition Lab' Day, 01 November, Paris, France
