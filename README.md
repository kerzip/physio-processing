# physio-processing

Collection of processing steps to analyse physiological data recordings.

One recording data stream includes [labstreaminglayer](https://github.com/labstreaminglayer) streams from following apps:
- [BrainVisionRDA](https://github.com/labstreaminglayer/App-BrainProducts/tree/8373006a351a1f84d82f598ce0abb723caac16d8/BrainVisionRDA)
- [gUSBamp](https://github.com/labstreaminglayer/App-g.Tec/tree/1f35f71d0a70ced7ea4025409293587cce7102cb/g.USBamp)
- [AppPupilLabs](https://github.com/labstreaminglayer/App-PupilLabs)

Processing of data is partially done with [mne](https://mne.tools/stable/index.html). For more information see A. Gramfort, M. Luessi, E. Larson, D. Engemann, D. Strohmeier, C. Brodbeck, R. Goj, M. Jas, T. Brooks, L. Parkkonen, M. Hämäläinen, MEG and EEG data analysis with MNE-Python, Frontiers in Neuroscience, Volume 7, 2013, ISSN 1662-453X
